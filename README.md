# 大学赴任者家族帯同支援プログラム - ポータルサイト

## プロジェクト概要

大学に赴任する研究者とそのご家族（パートナー）のキャリアと生活を包括的にサポートするポータルサイトです。

### 主な対象者
- 研究者のパートナー
- 地域在住者（リアリティング・進路検討希望者）
- 妊娠・出産後の再就職希望者
- スキルを活かしたい方
- 自己成長を目指す方
- 地域生活に慣れたい方

### 運営
- **主体**: ValueGlow（株式会社バリューグロー）
- **連携**: network2010.org（名古屋100km圏内の歴史・文化・観光情報）

## 実装済み機能

### 1. トップページ（index.html）
- プログラム紹介とビジョン表示
- 12項目のナビゲーションメニュー
- 名古屋の5つの魅力紹介（network2010.org連携）
- 大学別カスタマイズ機能（ロゴ・テーマカラー）
- 固定サイドバーナビゲーション
- スムーススクロール機能
- レスポンシブデザイン

### 2. 新規登録ページ（registration.html）
- 詳細なユーザー情報入力フォーム
- **必須項目**:
  - 氏名
  - メールアドレス
  - 電話番号
  - 所属大学
  - 利用目的（1つ以上選択）
- **任意項目**:
  - 住所
  - 現在の職業
  - 興味分野
  - 自己紹介
- フォームバリデーション機能
- 個人情報保護方針への同意チェック

### 3. 個人情報保護方針ページ（privacy-policy.html）
- ValueGlowの正式な個人情報保護方針を8セクション構成で掲載
- 個人情報の取り扱い、利用目的、第三者提供、安全管理措置など
- パンくずリスト付き

### 外部連携
- **network2010.org**へのリンク統合:
  - 歴史ページ
  - 文化ページ
  - 観光ページ
  - グルメページ
  - YouTubeチャンネル

## カスタマイズ方法

### 1. カラースキームの変更
`css/style.css`の`:root`セクションで色を変更できます：

```css
:root {
    --primary-color: #E87722;    /* メインカラー */
    --secondary-color: #5FB878;  /* セカンダリカラー */
    /* その他の色も同様に変更可能 */
}
```

### 2. 大学ロゴの追加
各大学のロゴを追加する場合：

1. `images/logos/`フォルダにロゴ画像を配置
2. `index.html`の大学カードセクションを編集
3. 大学ごとのテーマカラーを`:root`に追加

### 3. コンテンツの編集
HTMLファイルを直接編集してコンテンツを更新できます。各セクションはコメントで明確に区切られています。

## 注意事項

### 現在未実装の機能
以下の機能はデモ実装のため、実際には動作しません：

1. **メール送信機能**: 登録フォーム送信時のメール送信
2. **データベース連携**: ユーザー情報の保存
3. **ログイン認証**: ログイン機能

これらの機能は、バックエンドシステムの構築後に実装予定です。

### 個人情報の取り扱い
現在はデモ版のため、フォームに入力された情報は保存されません。実運用時には、個人情報保護方針に基づいた適切な管理システムを構築します。

## デプロイ方法

### GitHub Pages
1. GitHubリポジトリを作成
2. すべてのファイルをリポジトリにプッシュ
3. Settings → Pages → Sourceで`main`ブランチを選択
4. 数分後、`https://[ユーザー名].github.io/[リポジトリ名]/`でアクセス可能

### Netlify
1. [Netlify](https://www.netlify.com/)にアクセス
2. "Add new site" → "Deploy manually"
3. プロジェクトフォルダをドラッグ&ドロップ
4. 自動的にデプロイされる

### その他のホスティングサービス
- Vercel
- Firebase Hosting
- AWS S3 + CloudFront
- など、静的サイトホスティングサービスであればどこでも利用可能
