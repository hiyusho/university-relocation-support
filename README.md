# 大学赴任者パートナー帯同プログラム - ポータルサイト

## プロジェクト概要

大学に赴任する研究者とそのご家族（パートナー）のキャリアと生活を包括的にサポートするポータルサイトです。

### 主な対象者
- 研究者のパートナー
- 地域在住者（リアリティング・進路検討希望者）
- 妊娠・出産後の再就職希望者
- スキルを活かしたい方
- 自己成長を目指す方
- 地域生活に慣れたい方

### 運営
- **主体**: ValueGlow（株式会社バリューグロー）
- **連携**: network2010.org（名古屋100km圏内の歴史・文化・観光情報）

## 実装済み機能

### 1. トップページ（index.html）
- プログラム紹介とビジョン表示
- 12項目のナビゲーションメニュー
- 名古屋の5つの魅力紹介（network2010.org連携）
- 大学別カスタマイズ機能（ロゴ・テーマカラー）
- 固定サイドバーナビゲーション
- スムーススクロール機能
- レスポンシブデザイン

### 2. 新規登録ページ（registration.html）
- 詳細なユーザー情報入力フォーム
- **必須項目**:
  - 氏名
  - メールアドレス
  - 電話番号
  - 所属大学
  - 利用目的（1つ以上選択）
- **任意項目**:
  - 住所
  - 現在の職業
  - 興味分野
  - 自己紹介
- フォームバリデーション機能
- 個人情報保護方針への同意チェック

### 3. 個人情報保護方針ページ（privacy-policy.html）
- ValueGlowの正式な個人情報保護方針を8セクション構成で掲載
- 個人情報の取り扱い、利用目的、第三者提供、安全管理措置など
- パンくずリスト付き

### 外部連携
- **network2010.org**へのリンク統合:
  - 歴史ページ
  - 文化ページ
  - 観光ページ
  - グルメページ
  - YouTubeチャンネル

## カスタマイズ方法

### 1. カラースキームの変更
`css/style.css`の`:root`セクションで色を変更できます：

```css
:root {
    --primary-color: #E87722;    /* メインカラー */
    --secondary-color: #5FB878;  /* セカンダリカラー */
    /* その他の色も同様に変更可能 */
}
```

### 2. 大学ロゴの追加
各大学のロゴを追加する場合：

1. `images/logos/`フォルダにロゴ画像を配置
2. `index.html`の大学カードセクションを編集
3. 大学ごとのテーマカラーを`:root`に追加

### 3. コンテンツの編集
HTMLファイルを直接編集してコンテンツを更新できます。各セクションはコメントで明確に区切られています。

## 🔊 音声読み上げ機能

すべてのストーリーページに**Web Speech API**を使用した音声読み上げ機能を搭載。

### 主な機能
1. **音声選択** - 複数の日本語音声から選択可能
2. **リアルタイムハイライト** - 読み上げ中の段落を視覚的に強調
3. **速度調整** - 0.5倍〜2.0倍の速度変更
4. **ブックマーク** - 読み上げ位置を保存・再開
5. **レスポンシブUI** - PC・スマートフォン完全対応

### 技術仕様
- **使用API**: Web Speech Synthesis API
- **対応ブラウザ**: Chrome, Edge, Safari（完全対応）/ Firefox（一部制限）
- **データ保存**: localStorage（ブックマーク機能）
- **特殊処理**: rubyタグの読み仮名自動処理

### 実装ファイル
- `js/speech-reader.js` - コア機能（SpeechReaderクラス）
- `css/style.css` - 音声コントロールUIスタイル
