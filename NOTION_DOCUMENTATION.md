## 📋 プロジェクト概要

### 目的
大学に赴任する研究者とそのご家族(パートナー)のキャリアと生活を包括的にサポートするポータルサイト

### 対象者
- 研究者のパートナー
- 地域在住者(リアリティング・進路検討希望者)
- 妊娠・出産後の再就職希望者
- スキルを活かしたい方
- 自己成長を目指す方
- 地域生活に慣れたい方

### 運営
- 主体: ValueGlow(株式会社バリューグロー)
- 連携: network2010.org(名古屋100km圏内の歴史・文化・観光情報)

---

## ✨ 主な機能

### 1. トップページ
- プログラム紹介とビジョン
- 12項目のナビゲーションメニュー
- 名古屋の5つの魅力紹介(network2010.org連携)
- 大学別カスタマイズ機能(ロゴ・テーマカラー)

### 2. 新規登録ページ
- 詳細なユーザー情報入力フォーム
- 必須項目: 氏名、メールアドレス、電話番号、所属大学、利用目的
- 任意項目: 住所、職業、興味分野、自己紹介
- バリデーション機能付き

### 3. 個人情報保護方針
- ValueGlowの正式な個人情報保護方針を8セクション構成で掲載
- 個人情報の取り扱い、利用目的、第三者提供、安全管理措置など

**※ 注意: 現在はデモ実装のため、以下の機能は未実装です**
- 実際のメール送信機能
- データベース保存機能
- ログイン認証機能

## 🚀 デプロイ方法

### GitHub Pagesを使用する場合

1. **GitHubリポジトリの作成**
   - GitHub.comにログイン
   - 新規リポジトリ作成: `university-relocation-support`
   - Public設定で作成

2. **ファイルのアップロード**
   - リポジトリページで「Add file」→「Upload files」
   - すべてのファイルをドラッグ&ドロップ
   - 「Commit changes」をクリック

3. **GitHub Pagesの有効化**
   - リポジトリの「Settings」タブをクリック
   - 左メニューから「Pages」を選択
   - Source: 「main」ブランチを選択
   - 「Save」をクリック

4. **URLの確認**
   - 数分後、`https://[ユーザー名].github.io/university-relocation-support/`でアクセス可能
   - このURLを上部の「🌐 デモサイトURL」に記載

### Netlifyを使用する場合(より簡単)

1. **Netlify.comにアクセス**
   - GitHubアカウントでログイン

2. **デプロイ**
   - 「Add new site」→「Deploy manually」
   - プロジェクトフォルダをドラッグ&ドロップ
   - 自動的にデプロイされる

3. **カスタムドメイン設定(オプション)**
   - Site settings → Domain management
   - 独自ドメインを設定可能
