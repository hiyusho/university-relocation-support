🎯 1. システム概要

書類管理システムとは？

大学赴任者家族帯同支援プログラムの応募書類を、オンラインで安全に管理できるシステムです。

主な機能

  • ✅ 履歴書、職務経歴書などの書類をアップロード
  • ✅ アップロードした書類の一覧表示
  • ✅ 書類のダウンロード
  • ✅ 不要な書類の削除
  • ✅ ダッシュボードで最新の書類を確認

対応ファイル形式

  • 📕 PDF（.pdf）
  • 📘 Word文書（.doc, .docx）
  • 🖼️ 画像（.jpg, .jpeg, .png）
⚠️ ファイルサイズ制限
1ファイルあたり最大 10MB までアップロード可能です。

📤 4. 書類のアップロード

アップロード画面へのアクセス方法

方法1: ダッシュボードから

  1. ダッシュボードの「応募書類」セクション
  2. 「➕ 新しい書類をアップロード」をクリック

方法2: サイドバーメニューから

  1. 左側のサイドバーメニュー
  2. 「📁 応募書類」をクリック
  3. 「新規登録」ボタンをクリック

方法3: 直接URL

https://hiyusho.github.io/university-relocation-support/documents-upload.html

アップロード手順

ステップ1: 書類の種類を選択

「書類の種類」のドロップダウンメニューから選択：

  • 📄 履歴書
  • 📄 職務経歴書
  • 📄 カバーレター
  • 📄 資格証明書
  • 📄 ポートフォリオ
  • 📄 その他

ステップ2: ファイルを選択

方法A: クリックして選択

  1. 「ファイル」エリアをクリック
  2. ファイル選択ダイアログが開きます
  3. アップロードしたいファイルを選択
  4. 「開く」をクリック

方法B: ドラッグ&ドロップ

  1. アップロードしたいファイルをデスクトップやフォルダから選択
  2. 「ファイル」エリアにドラッグ
  3. ドロップすると自動的に選択されます

ステップ3: 説明・メモを入力（任意）

「説明・メモ」欄に、この書類に関する補足情報を入力できます。

入力例：

  • 2026年1月最新版
  • ABC株式会社応募用
  • 英語版

ステップ4: アップロードを実行

  1. すべての必須項目が入力されていることを確認
  2. 「📤 アップロード」ボタンをクリック

ステップ5: 進行状況の確認

プログレスバーが表示され、アップロードの進行状況を確認できます：

⬇️ 6. 書類のダウンロード

ダウンロード方法

方法1: ダッシュボードから

  1. ダッシュボードの「応募書類」セクション
  2. ダウンロードしたい書類の「⬇️」ボタンをクリック

方法2: 書類一覧から

  1. 書類一覧ページを開く
  2. ダウンロードしたい書類の「⬇️ 開く」ボタンをクリック

ダウンロード後の動作

パターンA: 新しいタブで開く
PDF ファイルの場合、ブラウザの新しいタブで表示されます。タブ内で閲覧、印刷、保存ができます。
パターンB: 自動ダウンロード
Word ファイルや画像ファイルは、自動的にダウンロードフォルダに保存されます。ブラウザの下部にダウンロード通知が表示されます。

ダウンロードしたファイルの保存場所

OS 保存場所
Mac /Users/ユーザー名/Downloads/
Windows C:\Users\ユーザー名\Downloads\
💡 ダウンロードのヒント
  • ダウンロードされるファイル名は、アップロード時のファイル名と同じです
  • 複数回ダウンロードすると、「ファイル名 (1).pdf」のように番号が付きます
  • PDF はブラウザで直接閲覧できます。保存が不要な場合は、タブを閉じるだけでOKです

🗑️ 7. 書類の削除

⚠️ 重要な注意事項
  • 削除した書類は 復元できません
  • 削除する前に、必要なファイルかどうか確認してください
  • 必要に応じて、事前にダウンロードしてバックアップを取ることをおすすめします

削除方法

方法1: ダッシュボードから

  1. ダッシュボードの「応募書類」セクション
  2. 削除したい書類の「🗑️」ボタンをクリック
  3. 確認ダイアログが表示されます
  4. 内容を確認して「OK」をクリック

方法2: 書類一覧から

  1. 書類一覧ページを開く
  2. 削除したい書類の「🗑️ 削除」ボタンをクリック
  3. 確認ダイアログが表示されます
  4. 内容を確認して「OK」をクリック

削除確認ダイアログ

┌─────────────────────────────────────┐ │ 「履歴書_v3.pdf」を削除してもよろしい │ │ ですか？ │ │ │ │ この操作は取り消せません。 │ │ │ │ [キャンセル] [OK] │ └─────────────────────────────────────┘

削除後の動作

  1. 「書類を削除しました」というメッセージが表示されます
  2. 書類一覧が自動的に更新されます
  3. 削除された書類は表示されなくなります
💡 削除のヒント
  • 間違えて削除してしまった場合：システムからは復元できません。元のファイルが手元に残っている場合は、再度アップロードしてください
  • 削除をキャンセルする：確認ダイアログで「キャンセル」をクリック、または、ダイアログの外側をクリック

❓ 8. よくある質問（FAQ）

Q1. アップロードできるファイルサイズの上限は？

A: 1ファイルあたり 最大10MB までです。

それ以上のサイズのファイルをアップロードしようとすると、エラーメッセージが表示されます。

対処法：

  • PDF の場合：圧縮ソフトでファイルサイズを小さくする
  • 画像の場合：解像度を下げる、JPEG 形式で保存する
  • Word の場合：画像を圧縮する、不要なページを削除する

Q2. 何件まで書類をアップロードできますか？

A: 特に制限はありません。必要な書類をすべてアップロードできます。

ただし、同じ内容の書類を複数アップロードするのではなく、最新版のみを保管することをおすすめします。

Q3. アップロードした書類を他の人が見ることはできますか？

A: いいえ、できません。

セキュリティ機能により：

  • ✅ ログインしたユーザー本人のみが自分の書類を閲覧できます
  • ✅ 他のユーザーがあなたの書類を見ることはできません
  • ✅ プログラム管理者のみが必要に応じてアクセスできます

Q4. スマートフォンからも利用できますか？

A: はい、利用できます。

  • ✅ iPhone（Safari）
  • ✅ Android（Chrome）
  • ✅ タブレット端末

画面サイズに応じて、レイアウトが自動的に調整されます。

Q5. アップロードした書類を編集できますか？

A: システム内での直接編集はできません。

編集が必要な場合：

  1. 書類をダウンロード
  2. パソコンで編集
  3. 編集後のファイルを新しくアップロード
  4. 古いバージョンを削除

Q6. ファイル名は変更できますか？

A: 現在のバージョンでは、ファイル名の変更機能はありません。

回避策：

  1. パソコンでファイル名を変更
  2. 変更後のファイルをアップロード
  3. 古いファイルを削除

Q7. パスワードを忘れてしまいました

A: プログラム管理者にお問い合わせください。パスワードのリセット手続きをご案内いたします。

🔧 9. トラブルシューティング

問題1: ログインできない

症状：

  • メールアドレスとパスワードを入力してもログインできない
  • 「認証に失敗しました」というエラーが表示される

解決策：

  1. メールアドレスを確認：スペルミスがないか確認（大文字・小文字は区別されません）
  2. パスワードを確認：大文字・小文字は区別されます。半角英数字で入力されているか確認
  3. Caps Lock を確認：キーボードの Caps Lock がオンになっていないか確認
  4. パスワードをリセット：パスワードを忘れた場合は、管理者にお問い合わせください

問題2: ファイルがアップロードできない

症状：

  • 「アップロード」ボタンをクリックしてもエラーになる
  • プログレスバーが進まない

解決策：

  1. ファイルサイズを確認：10MB 以下か確認
  2. ファイル形式を確認：.pdf, .doc, .docx, .jpg, .png のいずれかか確認
  3. インターネット接続を確認：他のウェブサイトが開けるか確認
  4. ブラウザのキャッシュをクリア：Mac: Cmd+Shift+Delete / Windows: Ctrl+Shift+Delete
  5. 別のブラウザで試す：Chrome、Safari、Edge など

問題3: アップロードした書類が表示されない

症状：

  • アップロードは成功したが、書類一覧に表示されない
  • ダッシュボードに書類が表示されない

解決策：

  1. ページをリロード：Mac: Cmd+R / Windows: Ctrl+R
  2. ブラウザのキャッシュをクリア：全期間を選択してクリア
  3. ログアウト＆ログイン：一度ログアウトして、再度ログインしてください
  4. 時間を置いて確認：まれに反映に1-2分かかる場合があります

問題4: ダウンロードできない

症状：

  • 「開く」ボタンをクリックしても何も起こらない
  • 「リンクが見つかりません」というエラーが表示される

解決策：

  1. ブラウザのポップアップブロックを確認：ポップアップを許可する設定に変更
  2. 別のブラウザで試す：Chrome、Safari、Edge など
  3. ページをリロード：Mac: Cmd+R / Windows: Ctrl+R
  4. 管理者に連絡：上記で解決しない場合はお問い合わせください

問題5: 画面が正しく表示されない

症状：

  • レイアウトが崩れている
  • ボタンが表示されない
  • 文字が重なっている

解決策：

  1. ブラウザを最新版に更新：古いブラウザは対応していない場合があります
  2. 推奨ブラウザを使用：Chrome（推奨）、Safari、Edge、Firefox
  3. ブラウザのズームレベルを確認：100% に設定（Mac: Cmd+0 / Windows: Ctrl+0）
  4. キャッシュをクリア：Mac: Cmd+Shift+Delete / Windows: Ctrl+Shift+Delete
